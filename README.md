# paper_highdim_Bayes_benchmark

## Tools
- `build.sh` - uses latexdiff to build PDF with changes to the last commit or staged version of `main.tex`
- `AGENTS.md` - CODEX guidelines
- `MEMORY.md` - compressed context of previous prompts
- `PLAN.md` - short term goals plan, change as needed
- `scripts/install_latex_deps.sh` - installation of dependencies
- `scripts/*.py` - supplementary plots for the text, preliminary implementation
